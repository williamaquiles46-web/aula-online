# 🎓 Aula Online - ERP Educacional
Sistema de gestão escolar desenvolvido em Python utilizando o framework JustPy e conceitos de POO (Programação Orientada a Objetos).

## 🚀 Funcionalidades
- Gestão completa de Estudantes e Professores.
- Agendamento de Aulas e Mentorias com vínculos dinâmicos.
- Controle Financeiro e Lançamento de Avaliações.
- Dashboard interativo em tempo real.

## 🛠️ Tecnologias
- Python 3.x
- JustPy (Framework Web)
- Tailwind CSS (Estilização)
